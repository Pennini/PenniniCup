Mata-Mata

Máximo {{ knockout_max_points }} pts

Palpite não-empate: pontua pela posição do vencedor (mandante × visitante) e pelo placar.

  1. 1Placar exato + vencedor {{ scoring_config.knockout_exact_and_advancing }} pts
  2. 2Vencedor + gols do vencedor {{ scoring_config.knockout_advancing_and_winner_goals }} pts
  3. 3Vencedor + diferença de gols {{ scoring_config.knockout_advancing_and_diff }} pts
  4. 4Vencedor + gols do eliminado {{ scoring_config.knockout_advancing_and_loser_goals }} pts
  5. 5Apenas o vencedor {{ scoring_config.knockout_advancing_only }} pts
  6. 6Nenhum acerto 0 pts
Palpite de empate (ex: 1×1, 2×2): ganha {{ scoring_config.knockout_draw_prediction_points }} pts fixos quando o jogo real terminar empatado no tempo regulamentar (qualquer placar, inclusive jogos decididos por pênaltis). Se o real não for empate, 0 pts.

Em caso de pênaltis, a pontuação considera o placar do tempo regulamentar. Empate no tempo regulamentar conta como empate para o palpite — mesmo que um dos times avance por pênaltis. O time que avançou de fato (incluindo decisão por pênaltis) só é usado para o bônus de avanço de time (Tipo 1) e para o critério de desempate de classificados/vencedores.

Scoring posicional no mata-mata

O vencedor é avaliado pela posição na chave (mandante × visitante), não pelo time em si. Mesmo errando o time, você pontua pelo placar se acertar qual posição venceu.

Exemplo (palpite não-empate)

Você palpitou Brasil × Tunísia com 1×0. O jogo real é Marrocos × Tunísia. Se Marrocos ganhar 2×1, você pontua: acertou o vencedor pela posição (mandante ganhou) e o saldo de gols (+1). Brasil nem estava no jogo, mas a lógica posicional valeu.

Palpite de empate

Palpitou empate (ex: 1×1)? Ganha {{ scoring_config.knockout_draw_prediction_points }} pts fixos se o jogo real terminar empatado no tempo regulamentar (qualquer placar). Se o real não for empate, 0 pts. O time marcado como vencedor (winner_pred) ainda conta para o bônus de avanço de time (Tipo 1) e para o critério de desempate de classificados/vencedores.
